## T22 — Stream titles from disk in batches during Stage 2
git branch: `ai-agent/t22_stream_titles_stage2`

PRD reference: §7.4, §12 (Memory pressure during index build)

**Problem:** 
In `app/build_index.py::generate_embeddings()`, the codebase currently reads all non-empty titles from the titles file into an in-memory Python list (`titles`) before starting the batch embedding generation:
```python
    titles: list[str] = []
    with titles_path.open(encoding="utf-8") as fh:
        for line in fh:
            stripped = line.strip()
            if stripped:
                titles.append(stripped)
```
For English Wikipedia, this list contains 6-10 million strings, incurring a major RAM overhead of ~1-2 GB (due to list pointers and Python object wrapper overhead for millions of strings). This list is held in memory during the entire embedding process.

**Solution:**
Modify `generate_embeddings()` to stream the titles from the file in batches of `batch_size` using a generator or file iterator, rather than loading the entire file into memory.

### Reference implementation

A helper function to batch titles from a file path:
```python
from itertools import islice

def _stream_titles_in_chunks(path: Path, chunk_size: int):
    with path.open(encoding="utf-8") as fh:
        # Create an iterator that filters empty/whitespace lines
        cleaned_lines = (stripped for line in fh if (stripped := line.strip()))
        while True:
            chunk = list(islice(cleaned_lines, chunk_size))
            if not chunk:
                break
            yield chunk
```

Then in `generate_embeddings()`:
```python
    # Replaces the titles list load and while loop:
    batch_idx = 0
    progress_cb(0.0)
    
    for chunk in _stream_titles_in_chunks(titles_path, max(int(batch_size), 1)):
        embeddings = model.encode(
            chunk,
            normalize_embeddings=True,
            show_progress_bar=False,
        )
        
        n_in_batch = embeddings.shape[0]
        batch_begin = int(batch_idx * n_titles / num_batches) if num_batches else 0
        memmap[batch_begin : batch_begin + n_in_batch] = embeddings
        
        batch_idx += 1
        progress = batch_idx / max(num_batches, 1) if num_batches else 1.0
        progress_cb(progress)
        
        if batch_idx % 10 == 0 or batch_idx == num_batches:
            logger.info("Embedding progress: %d / %d batches", batch_idx, num_batches)
            
        del embeddings
```
